## 🔭 Odak alanım

Türkçe **legal-tech / kamu-hukuk bilgi akışı**:
adli ve hukuki verinin makine-okunabilir biçimde yapılandırılması, kural tabanlı haber
izleme ve dezenformasyona karşı **erken sinyal** üretimi. Amaç; doğrulama/karar
mekanizmalarına alternatif değil, onlara **şeffaf ve denetlenebilir veri/araç katmanı** sunmak.
